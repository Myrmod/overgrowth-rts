# Tier 1 tech:
## The Amuns:
- starting building: power plant(obelisk)- provies power and buffs/ upgrades
- power plant unlocks->ref obelisk , barracks , airfeild
- ref obelisks unlocks-> ref altars
- rax/airfeild unlocks-> factory, naval yard
- factory unlocks -> t2 tech structure or mcv upgrade to be determined

## The plants:
- starting building: vine spreader ( no power for this faction )
- vine spreader unlocks-> barracks, ref, need to be built on it
- ref unlocks--> nothing
- rax unlocks -> factory , airfeild, naval yard
- factory ,airfeild, naval yard unlock-> tech structure/ upgrade building

## The vine humans:
- starting building : power plant
- power plant unlocks-> barracks and ref
- ref unlocks -> factory
- barracks unlocks -> factory
- factory unlocks -> airfeild, t2 structure. navy
- airfeild unlocks -> nothing

## The corta vine:
- starting building: power plant
- power plant unlocks-> barrack , ref, factory
- rax,factory -> unlocks airfeild, navy
- factory unlocks -> t2 structure
